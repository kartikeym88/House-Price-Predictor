# House-Price-Predictor
# House Price Predictor using Linear Regression

## Overview

This project implements a Linear Regression model on the California Housing Dataset using Python and scikit-learn.

The project demonstrates the complete machine learning workflow:

* Data Loading
* Exploratory Data Analysis (EDA)
* Data Preprocessing
* Train-Test Split
* Model Training
* Model Evaluation
* Model Saving

## Dataset

California Housing Dataset from scikit-learn.

Features:

* MedInc
* HouseAge
* AveRooms
* AveBedrms
* Population
* AveOccup
* Latitude
* Longitude

Target:

* MedHouseVal

* Matplotlib
